# 大学生记录生活系统（v1）

这是一个轻量、可长期坚持的系统：

- **每天 10-15 分钟**
- 覆盖：学习、作息、饮食、运动、情绪、开销、反思
- 输出：可回顾、可统计、可复盘

---

## 一、你每天要做的事

### 1) 早上（2-3 分钟）
- 打开 `模板/每日记录模板.md`
- 写今天三件最重要的事（MIT）
- 确认课程安排与学习重点

### 2) 白天（课后随手记）
- 每节课后记 2-3 行：
  - 学了什么
  - 没听懂什么
  - 下次补什么
- 用我帮你把语音/图片快速转成 Markdown

### 3) 晚上（8-10 分钟）
- 填写当天总结：睡眠、饮食、运动、学习完成度、情绪状态
- 写一句“明天最关键的一件事”

---

## 二、目录与分工（结合你现有 mynote）

- `睡眠/YYYY-MM-DD.md`：睡眠时长、入睡/起床
- `饮食/YYYY-MM-DD.md`：三餐与加餐
- `运动/YYYY-MM-DD.md`：运动内容与时长
- `生活/YYYY-MM-DD.md`：课程、日常、状态
- `随想/YYYY-MM-DD.md`：情绪、反思、想法
- `新闻/YYYY-MM-DD.md`：国内2 + 国际8（已自动化）
- `系统/模板/`：所有模板
- `系统/日志/`：系统执行日志（可选）

---

## 三、每周与每月复盘

- 每周日：使用 `模板/每周复盘模板.md`
- 每月最后一天：使用 `模板/每月复盘模板.md`

复盘关注三件事：
1. 哪些习惯真的在变好
2. 哪些时间被浪费最多
3. 下周/月只改一个关键点

---

## 四、我可以帮你自动做的

- 口述→自动分发到各分类文件
- 纸质笔记拍照→OCR→结构化 Markdown
- 每天新闻汇总（国内2+国际8）
- 一键生成周报/月报（学习、健康、状态）

> 原则：先保证能坚持，再追求精细统计。
