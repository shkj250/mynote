# 晚间总结与次日提醒机制

## 目标
- 每晚 23:30 自动发送当日总结、写入 mynote，并询问明日计划。
- 次日早上 08:00 自动提醒执行计划。

## 自动任务
1. **23:30 当日总结任务**
   - 汇总：睡眠/饮食/运动/生活/随想/新闻（当日）
   - 生成并写入：`系统/日报/YYYY-MM-DD.md`
   - 完成后询问：明天最重要的 3 件事
   - 有文件改动时自动 git push

2. **08:00 计划提醒任务**
   - 提醒执行昨晚计划
   - 若未制定计划，提醒立刻写下今日 3 件 MIT

## 备注
- 时间均为 Asia/Shanghai。
